# 🔐 Password Maker & Checker

A simple and secure web application to **generate strong passwords** and **check their strength** in real time.

## ✨ Features

- 🔑 Random password generator
- 📏 Custom password length
- 🔠 Uppercase, lowercase, numbers & symbols options
- 🛡️ Password strength checker
- 📊 Entropy calculation
- ⚡ Instant validation
- 📱 Responsive design

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 🚀 Getting Started

```bash
git clone https://github.com/yourusername/password-maker-checker.git
cd password-maker-checker
```

Open `index.html` in your browser or run it with Live Server.
